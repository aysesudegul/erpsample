# Dashboard Veri Notları

Bu klasörde dashboard tarafı için hazırladığım CSV veri dosyaları var. Veriler satın alma, stok, satış, brüt kar ve yenileme önerisi başlıklarına ayrıldı.

## Veri Dosyaları

- `mevcut_stok.csv`: ürün bazlı stok durumu
- `kritik_stok.csv`: minimum stok altındaki ürünler
- `tedarikci_harcamalari.csv`: tedarikçi bazlı satın alma tutarları
- `en_cok_satan_urunler.csv`: satış miktarı ve satış geliri
- `brut_kar_marji.csv`: ürün bazlı brüt kar bilgisi
- `yenileme_onerileri.csv`: stok yenileme ihtiyacı

## Dashboard Mantığı

Dashboard'u stok özeti, satın alma analizi, satış ve karlılık, yenileme önerileri şeklinde düşündüm. MacBook'ta Power BI Desktop olmadığı için veri tarafını CSV olarak düzenledim; aynı veri yapısı Google Sheets veya Looker Studio tarafına da uygun.
